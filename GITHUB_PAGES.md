# Publicação no GitHub Pages

1. Faça login no GitHub e crie um novo repositório chamado `site-winit` (ou nome de sua preferência).
2. Envie os arquivos deste projeto (`index.html`, `style.css`, `script.js`, `README.md`) para o repositório.
3. No GitHub, acesse as configurações do repositório, vá em "Pages" e selecione a branch principal (geralmente `main`) e a pasta raiz (`/`).
4. O site estará disponível em `https://<seu-usuario>.github.io/<nome-do-repositorio>/`.

> Este site é institucional, sem dados sensíveis ou links de pagamento. Pronto para análise do Mercado Pago.
