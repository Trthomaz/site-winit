# Winit - Plataforma de Automação para Comunidades

Este é o site institucional da Winit, uma comunidade e plataforma colaborativa dedicada a organizar, automatizar e evoluir campeonatos, eventos e experiências digitais para gamers e entusiastas de tecnologia.

## Estrutura
- Página única (one page)
- Seções: Sobre, Missão, Como funciona, Diferenciais, Apoio/Contribuição, Contato, Depoimentos, Rodapé
- Visual moderno, responsivo e minimalista
- Cores: preto, branco, vermelho (destaque)

## Hospedagem
Pronto para ser publicado no GitHub Pages.

> **Atenção:** Este site não contém dados sensíveis, links de pagamento ou informações pessoais reais.
